# ಠ_ಠ.js

Replaces `console.warn` with less-vague name to clearly indicate looking at something with a judgemental face.

```html
<script src="ಠ_ಠ.js"></script>
<script>
  var user = "Tim";
  ಠ_ಠ(user);
  // Logs "Tim" to console as a warning.
</script>
```
